Lesson 1: The Basics of the Web and HTML

Web Works!

In this unit I learned about basics of the web which are: The web: collection of HTML Docs HTML: Hyper Text Mark-up Language which is the main type of document of the web URLs: a simple protocol to communicate with the browser, the Internet and the Servers HTTP Web Apps.

When a user searches for a website, in fact, the computer sends HTTP Request to a Server. The server then finds the HTML document which is apt for what is requested and sends it back to the user's computer. The web browser interprets the page sent and displays it on the computer screen.

HTML

HTML or Hyper Text Mark-up Language is made up of: Text Content Markup Refs to other docs Links to other pages

HTML Markup

It is composed of opening tag and closing tag. This is also referred to as element Some examples of HTML markups that I learned are: BOLD, Italic,...

HTML Attributes

This is the same as tags but has Attribute = "value" for example Anchor tag is used to link a text to a web page. Void tags: has no contents

HTML Docs

it includes doctype, title, content.

Lesson 3: Adding CSS Style and HTML Structure

How to Avoid Repetition

Some of the coding in a HTML is the same and therefore the programmers have to repeat the codes they have written many times. This may cause mistake and lead to errors.

CSS

CSS or Cascaing Style Sheets enables the programmers to have a control on the style of the HTML elements. If HTML is like a structure of a house, the CSS shows the layout or the design of the house. The idea of Cascading in CSS is known as inheritance and understanding it requires abstract thinking. CSS is a great tool to avoid repetition in coding. For example, you can make the rule that all headings must be blue and you code this only once in the CSS, as opposed to writing down blue everytime for every heading.
